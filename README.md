## ✨ Features

| 기능 | 설명 |
|------|------|
| 🎮 **픽셀 아트 씬** | 마리오풍 배경 + 캐릭터 걷기 스탑모션 |
| 🖱️ **레트로 커서** | 클래식 픽셀 마우스 포인터 |
| 📺 **CRT 글리치** | 랜덤 지지직 스캔라인 효과 |
| 🎨 **3가지 테마** | 블루 / 블랙 / 프리즘 배경 전환 |
| 🌏 **한/영 전환** | 전체 텍스트 즉시 번역 |
| 🖼️ **픽셀 갤러리** | 포트폴리오 작업 이미지 |
| ⌨️ **부트 시퀀스** | 3초 타이핑 로딩 애니메이션 |
| 🕹️ **코나미 코드** | ↑↑↓↓←→←→BA 이스터에그 |

## 🛠️ Tech Stack

- **HTML5** — 시맨틱 마크업
- **CSS3** — CSS Variables, Keyframe Animations, CRT Effects
- **JavaScript** — Canvas API, TreeWalker i18n, IntersectionObserver
- **No Frameworks** — 100% Vanilla
